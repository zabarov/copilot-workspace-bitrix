# Пример страницы авторизации Bitrix

Этот документ содержит пример страницы `login.php`, которая подключает стандартный Bitrix-хедер/футер и компонент авторизации. Используйте этот фрагмент как шаблон для боевого проекта. **Не нужно** добавлять файл `login.php` в корень Bitrix Copilot Workspace — держите его в конкретном продукте или продемонстрируйте код прямо в документации.

## PHP-пример

```php
<?php
// Пример страницы login.php для подключения компонента авторизации.
require $_SERVER['DOCUMENT_ROOT'] . '/bitrix/header.php';
$APPLICATION->SetTitle('Login');

$APPLICATION->IncludeComponent('myproject:auth.login', '', [
    // параметры компонента при необходимости
]);

require $_SERVER['DOCUMENT_ROOT'] . '/bitrix/footer.php';
```

## Как использовать

1. Скопируйте код в рабочий проект (например, в публичный раздел `/login.php` или в отдельный компонент) только если это действительно требуется.
2. Адаптируйте пространство имён компонента (`myproject:auth.login`) и параметры под свой модуль.
3. Не добавляйте этот файл в корень workspace — здесь он существует лишь как документированный пример.
